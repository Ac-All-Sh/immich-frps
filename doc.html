1. 项目概述Overview

frps-panel 是基于 fatedier/frp 二次开发的 frp 内网穿透可视化管理面板。提供 frps 服务端与 frpc 客户端的 Web 管理界面，支持 16 种 CPU 架构，免配置文件，一键安装。适用于 Immich 照片同步等大文件传输场景的深度优化。

frps-panel is a visual management panel for frp, forked from fatedier/frp. It provides Web-based management for both frps server and frpc client, supporting 16 CPU architectures with one-click installation. Deeply optimized for large file transfer scenarios such as Immich photo sync.

核心特性Key Features

监控仪表盘 · 智能优化引擎 · 带宽管理 · 防火墙 · 在线升级 · 国际化 (中/英)

Monitor Dashboard · Optimization Engine · Bandwidth Manager · Firewall · Online Upgrade · i18n (ZH/EN)

3. Linux 部署Linux Deployment

从 Releases 下载对应架构的二进制文件，赋予执行权限后安装为系统服务。

Download the binary for your architecture from Releases, grant execute permission, and install as a system service.

# amd64
wget https://github.com/attychen/immich-frps/releases/latest/download/acfrps_v0.69.1_attychen_linux_amd64
chmod +x acfrps_v0.69.1_attychen_linux_amd64
sudo ./acfrps_v0.69.1_attychen_linux_amd64 install

安装程序将交互式提示：绑定端口 → 管理后台端口 → 后台地址 → 用户名 → 密码。

Installer prompts: bind port → dashboard port → dashboard address → username → password.

8. Docker 部署Docker Deployment

# host 网络模式
docker run -d --name frps-panel --network host --restart unless-stopped \
  -v /etc/frp:/etc/frp -v /var/log/frp:/var/log/frp \
  acallsh/frps-panel:latest

# 桥接模式
docker run -d --name frps-panel --restart unless-stopped \
  -p 7000:7000 -p 7500:7500 -p 8080:8080 \
  -v /etc/frp:/etc/frp acallsh/frps-panel:latest

docker-compose.yml

version: '3.8'
services:
  frps-panel:
    image: acallsh/frps-panel:latest
    network_mode: host
    restart: unless-stopped
    volumes:
      - /etc/frp:/etc/frp
      - /var/log/frp:/var/log/frp

10. 客户端配置Client Configuration

frpc 客户端需在 frps Web 面板中生成，二进制内嵌认证密钥。

frpc clients are generated via the frps Web panel. The binary embeds the auth key.

  1. 登录 frps Web 面板 → 用户管理 → 新增用户
  2. Login frps Web Panel → User Management → Add User
  3. 选择用户 → 生成客户端 → 下载对应架构 frpc
  4. Select User → Generate Client → Download frpc for target arch
  5. 上传至目标设备，执行 ./acfrpc_xxx install
  6. Upload to target device, run ./acfrpc_xxx install

13. 常见问题FAQ

Q: 如何确认设备架构？

Q: How to check architecture?

uname -m
# x86_64 → amd64 / aarch64 → arm64 / armv7l → armhf

Q: 面板端口无法访问？

Q: Panel port not accessible?

sudo ufw allow 7500
# or
sudo firewall-cmd --add-port=7500/tcp --permanent && sudo firewall-cmd --reload

Q: 如何升级？

Q: How to upgrade?

Web 面板 → 在线升级，或重新下载最新二进制覆盖安装。

Web Panel → Online Upgrade, or re-download the latest binary.
